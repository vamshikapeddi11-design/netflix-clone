# Netflix Clone 🎬

A responsive Netflix-inspired web application built using HTML, CSS, and JavaScript. The project features a modern UI with movie categories, search functionality, smooth scrolling, and login/sign-out pages.

## 🚀 Features

* Netflix-style UI
* Movie search functionality
* Responsive design
* Smooth scrolling
* Login & Sign Out pages

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
